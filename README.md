SkipDich
SkipDich ist eine WebApp zum Tracken von meinem Seilspring-Training.
Die Applikation speichert alle Daten lokal im Browser und wurde im Rahmen der Ausbildung als Informatiker EFZ (Applikationsentwicklung) entwickelt für das Modul 245. 

Live-Version:
https://asmrrax-star.github.io/skipdich/

Funktionen

Training erfassen (Datum, Sprünge, Dauer, Übungstyp, Intensität, Stimmung)
Dashboard mit Statistiken und Chart der letzten 10 Sessions
Persönlicher Rekord wird automatisch markiert
Streak-Anzeige bei mehreren Trainingstagen in Folge
Verlauf aller Sessions mit Löschfunktion
Keine Internetverbindung nötig, alles läuft komplett im Browser

Lokale Ausführung
Repository klonen:
git clone https://github.com/asmrrax-star/skipdich.git
Danach die Datei index.html im Browser öffnen.
Es ist kein Build-Prozess oder zusätzliche Software erforderlich.

Ziel des Projekts

Ziel war es, eine Applikation zu entwickeln, die gleichzeitig modulrelevant ist
und einen persönlichen Mehrwert bietet. SkipDich entstand aus dem eigenen Bedarf,
meine Seilspring-Trainings einfach und übersichtlich zu tracken, ohne unnötige
Komplexität, ohne Backend, direkt im Browser nutzbar.
